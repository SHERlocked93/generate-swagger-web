# generate-swagger-web

fork 自[Journey98/generate-swagger-web](https://github.com/Journey98/generate-swagger-web)
，他这个只能单个接口的生成，感觉不太方便，改成了可以生成模块所有的接口的形式

## 运行

```shell
live-server
```

然后在文档接口中输入 swagger 在线链接，即可输出结果
